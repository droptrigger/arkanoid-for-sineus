## Игра в жанре Arkanoid для Хакатона Синеус 2024

### Запуск

Чтобы запустить игру, необходимо выполнить испольняемый файл `Build/Arkanoid.exe`.
Он распакует необходимые для игры файлы во временную папку и запустит игру.
